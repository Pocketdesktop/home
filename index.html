Pocket Desktop

What is pocket desktop?

Taking desktop in pocket. Pocket desktop is a server based android application in which you can get control of your whole desktop within the app just as a remote for a tv. This application is used to control the computer in various ways.

Features

File

×

FILE FEATURES

Each file of your desktop can be accessed through this feature.

Attributes such as file size, name, etc. can also be viewed.

Switching between all the running application can also be done


Desktop

×

Desktop FEATURES

The system desktop will be projected on android device screen

The touch display of the android will be used to control the system.

Long pressing on android screen does right click on the desktop screen.

The application also supports multi gesture such as pinch and zoom.


Music

×

MUSIC FEATURES

This feature can be used to control media players running on the desktop.

The control pad is provided with all the keys usually used in media players.

Pressing these keys on the android display sends data to the desktop application which accordingly controls the media player.


Game

×

GAME FEATURES

This feature enables the android device to be used as a joystick for games.

Left or right tilt of the android are used in moving left and right motion

A S W D should be done in the game setting to control the direction.
